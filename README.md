# JSON-como-DB

Nodejs, Express &amp; JSON como DB

## Introducción 

Esta aplicación web se guardan datos de libros con su imagen y tambien se eliminan, el guardado de los datos es de forma local en un `.json `. 

## Tecnologias

* `Express`
* `morgan`
* `bootstrap`
* `uuid`
* `middlewere`
* `nodemon`
* `node.js`
* `HTML`
* `CSS`
